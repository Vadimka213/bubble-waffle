 Bubble Waffle Website
Опис проєкту

Bubble Waffle — це односторінковий вебсайт для онлайн-замовлення гонконгських вафель.
Проєкт демонструє базові навички HTML та структурування контенту.

Сайт містить інформацію про продукт, меню, процес замовлення та форму для оформлення заявки.
